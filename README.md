# My dotfiles

Install my vimrc

```shell
wget https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/master/.vimrc
curl https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/master/.vimrc
vim
```

Install my .tmux.conf

```shell
wget https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/master/.tmux.conf
curl https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/master/.tmux.conf

For before version <=2.8
wget https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/tmux-2/.tmux.conf
curl https://raw.githubusercontent.com/ronnypolloqueri/dotfiles/tmux-2/.tmux.conf
```
